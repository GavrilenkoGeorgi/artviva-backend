### Build Setup

``` bash
# install dependencies
npm install

# start server
npm start

# nodemon watch
npm run watch

# run tests
npm run test

# run api test
npm run test:userapi

#build react user ui
npm run build:ui

#build ui and deploy
npm run deploy:full

#start in a test environment mode
npm run start:test
```
